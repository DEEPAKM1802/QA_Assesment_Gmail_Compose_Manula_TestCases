Gmail Compose Functionality - Test Case Repository


Overview
  This repository contains a comprehensive set of manual test cases for the Gmail Compose functionality. 
  The test cases are designed to validate the behavior, usability, reliability, accessibility, and compatibility of the compose email workflow in Gmail.


The test suite includes:
  Traditional manual test cases
  Gherkin-style BDD test cases
  Positive and negative test scenarios
  UI and usability validations
  Recovery and compatibility checks
  Boundary and validation test cases

  All test cases are documented in an Excel sheet for easy review and execution.


Gherkin / BDD Coverage
  Each traditional manual test case includes a corresponding Gherkin-style BDD scenario.
  This helps bridge manual testing with behavior-driven testing practices and future automation efforts.


Tools Used
  Microsoft Excel
  Manual Testing Techniques
  BDD / Gherkin Syntax
  Objective


The objective of this repository is to provide:
  Structured manual test coverage
  Clear and reusable test documentation
  Readiness for future automation implementation
  Better traceability between traditional and BDD-style testing
